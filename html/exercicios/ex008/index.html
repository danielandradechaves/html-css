Listas Ordenadas

A ordem da lista faz diferença, OL.

  1. Acordar
  2. Ligar o computador
  3. Tomar café
  4. Escovar os dentes
  5. Estudar HTML

Quando se tem uma lista enumerada, como o exemplo acima, se for preciso acrescentar um item no meio dela, teria que reenumerar todos os outros itens para não ficar bagunçado. Então essas tags, fazem a enumeração de forma automatica

A tag OL tem um parâmetro chamado (type=""), onde você pode escoher se a lista será numérica(1, 2, 3), em ordem alfabética com letras maiúsculas(A, B, C) ou minúsculas(a, b, c) e tambem em algarismos romanos com letras maiúsculas(I, II, III) ou minúsculas(i, ii, iii)

Listas não Ordenadas

A ordem da lista não faz diferença, UL.

  • catchup
  • mostarda
  • só

A tag UL também tem o parâmetro (type=""), com as opções: disc(um ponto preto), circle(um circulo não preenchido), squere(quadrado)
